# intent-check スキル

## 概要

ユーザーが提示した素材（スクリーンショット・ラフスケッチ・テキスト・コードなど）から AI が推論した「ユーザーの意図」を**不可分な単位**に分解し、1件ずつ齟齬がないか確認するスキル。

確認前に推論内容をマークダウンファイルとして保存し、齟齬が発覚するたびにそのファイルをリアルタイムで修正する。

## 実行手順

### フェーズ1: 素材の確認

現在の会話コンテキスト（直前のメッセージ）を対象とする。
画像・スクリーンショット・テキスト・コードなど、ユーザーが提示したすべての素材を入力として扱う。

素材が見当たらない場合はユーザーに提示を求める:

```
確認したい素材（スクリーンショット・説明文・コードなど）を貼り付けてください。
```

### フェーズ2: 意図の推論と分解

素材をもとに以下の観点から推論する:

- **What**: 何を作りたいか／何を達成したいか
- **Why**: なぜそれが必要か（目的・背景）
- **How**: どのような手段・方法を想定しているか
- **Who**: 誰が使うか（対象ユーザー・操作者）
- **Constraint**: 制約・前提条件（技術・期間・規模など）

推論結果を**不可分な推論単位**に分解する。

**分解の基準:**
> **「この推論は合っていますか？」という問いに y/n で答えられる単位**

一つの推論が複数の判断を含んでいたら、さらに細かく分割する。

### フェーズ3: 推論ファイルの保存

`~/.claude/intent-check/` に推論内容をマークダウンファイルとして保存する（`Write` ツール）。

**ファイル名形式:** `{形容詞}-{名詞}-{動詞}.md`（ランダムな英単語3語）
例: `curious-block-emerge.md`

**ファイルフォーマット:**

```markdown
# 意図推論: {素材の簡易説明}

作成日時: {YYYY-MM-DD HH:MM}

## 素材の概要

{素材の種類と内容を簡潔に記述}

## 推論一覧

| # | 観点 | 推論内容 | 確認結果 |
|---|------|----------|----------|
| 1 | What | ... | 未確認 |
| 2 | Why  | ... | 未確認 |
| ... | | | |

## 修正履歴

（確認中に変更が生じたら追記）
```

保存後、ファイルパスをユーザーに伝える:

```
推論内容を保存しました: ~/.claude/intent-check/curious-block-emerge.md
一つずつ確認します。
```

### フェーズ4: ステップバイステップ確認

#### 全件一覧の表示

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
意図確認: {素材の簡易説明}（全N件）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. [What] ...
  2. [Why]  ...
  ...
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
y で合っている / その他で訂正・中断
```

#### 各推論の確認

```
確認 N/X

[観点]: [推論内容]
根拠: [素材のどの部分からそう推論したか]

この推論は合っていますか？
```

**フォーマット原則:**
- 推論内容は具体的に書く（「UIを作りたい」ではなく「デスクトップアプリのメイン画面を作りたい」のように）
- 根拠は必ず示す（「図の左上にある矢印から推論」など）
- 進捗を常に表示: "確認 N/X"

#### 入力のハンドリング

| 入力 | 動作 |
|------|------|
| `y` / `yes` / `はい` / `合ってる` | 承認 → ファイルの確認結果を「承認」に更新 → 次へ |
| `n` / `no` / `いいえ` / `違う` | 齟齬あり → 正しい内容を聞く → 修正案を提示 → 再確認 |
| 自由記述 | 訂正・補足として扱う → 修正案を提示 → 再確認 |
| `q` / `quit` / `中止` | 中断 → その時点までの集計を表示 |

#### 齟齬発覚時のファイル即時更新

齟齬が承認された場合、**その場で `Edit` ツールで推論ファイルを更新する**:
- 推論一覧テーブルの該当行を修正
- 確認結果を「齟齬あり → 修正済み」に更新
- 修正履歴セクションに変更内容を追記

次の推論の確認に進む前に書き込みを完了させること。

#### 全件完了後の集計表示

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
確認完了: 一致 N件 / 訂正 N件
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
訂正した推論:
  #N [元の推論] → [修正後]

推論ファイルを更新しました: ~/.claude/intent-check/curious-block-emerge.md
意図確認が完了しました。
```

## 注意事項

- **コード実装・タスク実行はしない**: 意図の推論と確認のみ行う
- **使用ツール**: `Read`, `Glob`, `Edit`, `Write` のみ
- **変更は即時反映**: 齟齬が確定したら、次の推論に進む前に `Edit` でファイルを更新する
- **一件ずつ表示**: 複数の推論をまとめて確認しない
- **推論の粒度**: 「y/n で答えられる」単位まで細かく分割する。細かすぎる場合は同一観点内でグループ化してもよい
- **不明な観点はスキップ**: 素材から推論できない観点（例: 素材にユーザー情報がなければ Who）は無理に推論せず省略する
