# asw-utils

Claude Code の開発体験を向上させるユーティリティスキル集。

## スキル一覧

| スキル | 呼び出し | 概要 |
|--------|---------|------|
| `wantree` | `/asw-utils:wantree` | ヒアリング形式で要件定義YAMLを作成・管理 |
| `plan-review` | AUTO-TRIGGER のみ | プランを最小決定事項に分解してステップバイステップ確認 |

## インストール

```
/plugin install https://github.com/inadati/asw-utils
```
